== 栈和队列

=== 找到栈中最小值

维护两个Stack:
一个dataStack, 负责正常数据压栈出栈, 一个minStack, 负责在栈顶维护栈的最小值.

==== 算法1

* *压栈时:* 如果minStack为空或者要压入的元素小于等于最小值, 则将这个元素压入minStack.
* *出栈时:* 如果出栈的元素小于等于最小值, 则minStack出栈.

==== 算法2

* *压栈时:* 如果minStack为空或者要压入的元素小于等于最小值, 则将这个元素压入minStack, 否则, 压入最小值.
* *出栈时:* minStack直接弹出.
