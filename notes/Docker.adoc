== 容器

 容器允许在一台机器上运行多个服务, 这些服务运行在宿主机的操作系统上. 容器不仅为各个服务提供不同的环境, 并且能将他们相互隔离.
 每个虚拟机需要有自己的操作系统, 消耗了额外的系统资源. 而一个容器只是运行在宿主机上被隔离的单个进程.
 容器使用namespace隔离进程, 使用cgroup限制系统资源消耗.
