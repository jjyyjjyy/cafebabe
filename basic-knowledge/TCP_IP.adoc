== 2. 数据链路层

=== 1. CSMA/CD

`一个站点首先检测网络上正在发送的信号, 并在网络空闲时发送自己的帧.
如果其他站点碰巧同时发送, 则为一次碰撞. 每个站点等待一个随即时间, 然后再次尝试发送.
随后如果再次碰撞, 等待的时间翻倍. 最终, 每个站点会得到机会发送, 或者在尝试一定次数(16)后超时.
`
